从公开来源，自动导入真实岗位。

这个页面会调用 Vercel API，从公开职位 API 和公司 ATS 页面拉取岗位，筛选 legal / compliance / AML / KYC / Web3 / cross-border 相关内容，然后写入 Supabase。

导入设置

先用 Remotive 测试，不需要 API key。Adzuna 需要在 Vercel 设置 ADZUNA_APP_ID 和 ADZUNA_APP_KEY。

查看职位页 查看搜索页
准备好了。建议先用 Remotive 测试一次。
注意：Remotive 要求展示来源并回链。导入后的岗位会保存 source_name 和 source_url。不要抓取 LinkedIn / Indeed / BOSS 这类反爬和条款风险高的平台。

导入结果

导入成功后，职位会写入 Supabase 的 platform_items，然后首页搜索和职位页会自动读取。
